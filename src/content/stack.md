# tools & stack

Things I use every day and trust to ship real work.

---

## languages

- **TypeScript** — everything frontend and most backend
- **Python** — data processing, ML pipelines, scripting
- **SQL** — Postgres, mostly through Supabase

---

## frameworks

- **Next.js** — App Router, RSC, the whole thing. My default for everything web.
- **React** — goes without saying
- **Tailwind CSS** — utility-first, never going back

---

## infra

- **Vercel** — deploy Next.js apps, zero config
- **Supabase** — Postgres + auth + realtime + storage. Replaces a lot of backend work.
- **Linux VPS** (Hostinger) — always-on processes, agents, background jobs

---

## AI

- **Claude** — primary LLM for coding, reasoning, agents
- **OpenAI** — embeddings, specific use cases
- **Claude Code** — daily driver for writing code

---

## tools

- VS Code / Cursor
- GitHub
- Figma (for when design matters)
- iTerm2, zsh

---

## philosophy

Pick boring tech when it works. Reach for new tools when they solve a real problem. Ship constantly. Refactor when the code tells you to.
